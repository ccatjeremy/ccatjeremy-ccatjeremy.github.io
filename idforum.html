About NCKUID Forum
ID forum is a website for NCKU students changing some industrial design information. ID forum banner style is an interesting project we did when my teammate and I held leaders of webmaster group. New banner used flat design to make the "ID2" clear, and the color made the forum simple. We would design some fun banners for celebrating when festivals coming.
